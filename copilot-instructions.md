---------------------------------------
🎯 เงื่อนไขสำคัญของระบบ (VERY IMPORTANT)
---------------------------------------

ระบบนี้รองรับการจำแนกเมล็ดข้าว “เฉพาะ 5 สายพันธุ์เท่านั้น” ได้แก่:

1. เมล็ดข้าวหอมมะลิ 105
2. เมล็ดข้าวกล้อง
3. เมล็ดข้าวเสาไห้
4. เมล็ดข้าวเหนียว กข6
5. เมล็ดข้าวเหนียวดำ

❗ กฎสำคัญ:
- ถ้าผลลัพธ์จาก Roboflow ไม่อยู่ใน 5 สายพันธุ์นี้
  → ให้แสดงว่า: "ไม่สามารถจำแนกสายพันธุ์นี้ได้"
- ถ้า confidence ต่ำกว่า 0.70
  → ให้แสดงว่า: "ความมั่นใจต่ำ ไม่สามารถระบุสายพันธุ์ได้"
- ห้ามเดาสายพันธุ์เองเด็ดขาด
- ต้อง validate class ที่ได้จาก API ทุกครั้งก่อนแสดงผล

---------------------------------------
🎯 การทำงานของระบบ
---------------------------------------

ระบบต้องสามารถ:
- อัปโหลดรูปเมล็ดข้าว
- แสดง preview
- ส่งไป Roboflow API
- รับค่า JSON
- ตรวจสอบ class + confidence
- แสดงผลลัพธ์อย่างถูกต้องตามกฎ
- บันทึกผลลงฐานข้อมูล

---------------------------------------
🎯 รูปแบบการแสดงผลลัพธ์
---------------------------------------

กรณีปกติ:
- ชื่อสายพันธุ์ (ภาษาไทย)
- ค่า confidence (%)
- แสดงเป็น card สีเขียว

กรณีไม่รองรับ:
- แสดง card สีแดง
- ข้อความ:
  "ไม่สามารถจำแนกสายพันธุ์นี้ได้"

กรณี confidence ต่ำ:
- แสดง card สีเหลือง
- ข้อความ:
  "ความมั่นใจต่ำ ไม่สามารถระบุสายพันธุ์ได้"

---------------------------------------
🎯 Mapping class จาก Roboflow → ภาษาไทย
---------------------------------------

ต้อง map ค่า class เป็นภาษาไทย เช่น:

- jasmine_rice_105 → เมล็ดข้าวหอมมะลิ 105
- brown_rice → เมล็ดข้าวกล้อง
- rd15 → เมล็ดข้าว กข15
- rd6_sticky → เมล็ดข้าวเหนียว กข6
- black_sticky_rice → เมล็ดข้าวเหนียวดำ

(สามารถปรับตามชื่อ model จริงได้ แต่ต้องมี mapping เสมอ)
